impasse-tool
============

# Build

## Build a package

    $ mvn package

## Copy dependencies

    $ mvn dependency:copy-dependencies

# Report

## Generate a report

    $ bash scripts/report.sh -n [project name] -p [plan name] -r -o [Output File]

# Import

## From TestLink

    $ bash scripts/testlink-import.sh [project name] [TestLink XML File]
